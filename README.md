# Sales Performance KPI Dashboard (Power BI Portfolio Project)

This project showcases an end-to-end Power BI dashboard built to analyze sales, profitability, regional performance, and customer insights. It demonstrates my ability to clean and model data, create DAX measures, and design interactive visualizations that support informed business decisions.

## Overview

The dashboard provides a complete view of company sales across regions, years, customer types, and product segments. It highlights trends, profitability metrics, and customer-level performance to help stakeholders monitor targets and identify growth opportunities.

## Objectives

- Transform raw Excel sales data into a structured and reliable data model.

- Build KPIs such as Total Sales, Gross Profit, Net Profit, GM%, and NP%.

- Create interactive visuals to support drill-down analysis.

- Provide users with flexible filtering (year, month, region, customer type).

## Key Insights & Features

### Sales Overview

- Total Sales vs Target performance

- Gross Profit and Net Profit analysis

- Sales by Country and Subcategory

### Profitability Analysis

- GM% and NP% trends

- Profitability by City and Year

- Region-wise performance comparisons

### Change Analysis

- Year-over-year customer-level sales changes

- Growth vs decline segmentation

### Customer Analysis

- Customer segmentation by Hypermarket vs Supermarket

- Detailed metrics table including Sales LY, Target, Profit, GM%

- Drill-down capabilities for deeper insights

## Tools & Techniques

- **Power BI** Desktop for visualization and modeling

- **Power Query** for data cleaning and transformation

- **DAX** for KPI and calculated measures

- **Excel** as the initial data source
